# AI Smart Job Matching Engine 🚀

An AI-powered job matching platform that intelligently matches job seekers with opportunities based on their skills and desired roles.

## Features ✨

- **AI Job Matching:** Smart algorithm matches jobs based on user skills
- **User Authentication:** Secure registration and login
- **Save Jobs:** Bookmark favorite job listings
- **Job Applications:** Apply for jobs with cover letters
- **Application Tracking:** Monitor application status
- **Profile Management:** Update skills and preferences
- **Real-time Scoring:** Match percentage for each job

## Tech Stack 💻

- **Backend:** Python, Flask
- **Database:** SQLAlchemy (SQLite/PostgreSQL)
- **Frontend:** HTML, CSS, Bootstrap 5, JavaScript
- **Authentication:** Flask-Login
- **Deployment:** Vercel
